# Оркестратор аутентификации

Spring Boot микросервис (WebFlux), отвечающий за оркестрацию процессов аутентификации пользователей в системе.
Не имеет собственной базы данных – все данные о пользователях (учётные записи, пароли, роли) хранятся в Keycloak

Все входные и выходные модели (DTO) генерируются из OpenAPI YAML-файла с помощью плагина "openapi-generator".
Генерация происходит автоматически при сборке проекта Gradle.
Сгенерированные классы помещаются в build/generated/individuals_api_v1.0

### Функциональные возможности
    POST /v1/auth/registration — регистрация нового пользователя (без авторизации)
    POST /v1/auth/login — аутентификация пользователя по email и паролю (без авторизации)
    POST /v1/auth/refresh-token — обновление access/refresh токена (без авторизации)
    GET /v1/auth/me — получение информации о текущем пользователе (требуется Bearer Token)

### Observability

* /actuator/prometheus подключён к Prometheus
* Logback в JSON-формате. Логи доступны в Grafana через Loki
